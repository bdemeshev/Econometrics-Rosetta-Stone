Данные содержат информацию о количестве рыбы, пойманной людьми на отдыхе.

Camper - наличие/отсутствие палатки. Child - количество детей, которых взяли на рыбалку. Persons - количество людей в группе. Count - количество пойманной рыбы

Строим Пуассоновскую регрессию. AIC, который мы видим в описательных статистиках, рассчитан по следующей формуле: \(AIC = -2log(L) + 2k\) \(AIC = -2log(L) + klog(N)\)

где L — значение функции правдоподобия модели, k — количество переменных, N — число наблюдений.

glm count camper child persons, family(poisson)

Можем посчитать AIC и BIC по другой формуле и получить результат, аналогичный выводу в R. \(AIC = \frac {-2log(L) + 2k}{N}\)
